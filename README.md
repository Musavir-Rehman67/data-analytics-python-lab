# data-analytics-lab
# 📊 Data Analytics Using Python Lab

## Overview

This repository contains Python programs, notebooks, datasets, and a mini project developed as part of the **Data Analytics Using Python Laboratory**. The lab covers the complete data analytics workflow, including data preprocessing, visualization, statistical analysis, machine learning, clustering, dimensionality reduction, time series forecasting, and an end-to-end analytics pipeline.

The repository is designed to help students understand both the theoretical concepts and practical implementation of modern data analytics techniques using Python.

---

# Course Outcomes

After completing this laboratory, students will be able to:

- Apply Python-based tools to perform data acquisition, exploratory data analysis (EDA), and preprocessing techniques for structured datasets.
- Analyze datasets using statistical methods including hypothesis testing, correlation analysis, and ANOVA.
- Design and implement regression and classification models and evaluate their performance using appropriate statistical and machine learning metrics.
- Apply unsupervised learning techniques such as clustering and dimensionality reduction to discover hidden patterns in data.
- Develop complete end-to-end data analytics pipelines including forecasting, model evaluation, and interpretation for solving real-world problems.

# Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Data-Analytics-Using-Python.git
```

Navigate to the project directory:

```bash
cd data-analytics-python-lab
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
